# Style Guide

This document defines the general formatting standards for all
AI-generated
content within the Strategy Garden.

## **Markdown Formatting Rules**

- **Empty Lines After Headings**: Always include exactly one empty line between
  a heading and its subsequent content.

- **Empty Lines Between Block Elements**: Always include exactly one empty line
  between paragraphs and list items.

- **Empty Lines Between List Items**: Always include exactly one empty line
  between individual list items.

- **Maximum Line Length**: Always wrap lines to a maximum of 65 characters. Use
a formatter like `prettier --prose-wrap always --print-width 65`
to adhere to
  this standard.
